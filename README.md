# ☁️ Cloud-Based File Storage System

## 📌 Project Overview
This project is a cloud-based file storage system developed using AWS services. It allows users to upload, store, and retrieve files securely using cloud infrastructure.

## 🎯 Objectives
- To understand cloud storage architecture
- To implement file storage using AWS S3
- To ensure secure access using IAM policies
- To demonstrate scalability of cloud systems

## 🚀 Features
- Upload files to cloud
- Download files anytime
- Secure storage using AWS
- Scalable and reliable system

## 🛠️ Technologies Used
- AWS S3 (Storage)
- AWS IAM (Security)
- Python (Flask)
- HTML, CSS

## 🧩 System Architecture
User → Web Interface → Backend Server → AWS S3 Bucket


## 📂 Project Structure
- src/: Backend code
- templates/: HTML files
- static/: CSS files
- screenshots/: Output images
- report/: Final report

## 📈 Future Scope
- Add user authentication
- Implement file encryption
- Develop mobile application
